Started learning `svelte` from basics

## installation

```bash
npm create svelte@latest myapp
cd myapp
npm install
npm run dev
```

## Why svelte is faster than react ?

`svelte` is a compiler that compiles `.svelte files` into optimized javascript files at `build time` rather than interpreting code at `run time`.
so most of the work are done in build time instead of in client browser.

## svelte structure

- in .svelte files we can write HTML,CSS (scoped) and JS
- scripts we need are added inside `script tag` and styles are inside `style tag`
- styles are `scoped` in nature

eg:

```svelte
<script>
let name = "hi !!"
</script>

<h1> {name} </h1>

<style>
h1 {
    color: red
}
</style>
```

## Event listeners

- event listeners are added using `on:[ event ]` on the HTML tag

```svelte
<button on:click={} > Click </button>
```

## passing props

to receive a prop we need to add `export` keyword at the script tag.

```svelte
// Child.svelte

<script>
export let data;
</script>
<h1> {data} </h1> // 11

// App.svelte

<script>
import Child from "./Child.svelte";
</script>
<Child data={11} />
```

## Logical rendering

svelte has `if/ else if / else` block.

```svelte
{#if count > 10}
	<p>{count} is greater than 10</p>
{:else if count < 5}
	<p>{count} is less than 5</p>
{:else}
	<p>{count} is between 5 and 10</p>
{/if}
```
