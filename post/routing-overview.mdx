## Routing Overview

- `react-router-dom` package for page routing.
- wrap the App.jsx with `BrowserRouter` and `Routes` tag.
- Add `Route` tag to show matching component

```jsx
<BrowserRouter>
  <Routes>
    <Route path="/" element={<Homepage />} />
    <Route path="/product" element={<Product />} />
    <Route path="/pricing" element={<Pricing />} />
    <Route path="*" element={<PageNotFound />} />
  </Routes>
</BrowserRouter>
```
- all the router related tags have to be placed inside `BrowserRouter` tag
