## Custom hooks

- when we need to reuse the non-visual logics
- when those logic need one or more other hooks then use custom hooks else use normal functions outside the component.

## Rules for custom hooks

- all the rules of hooks applicable for custom hooks
- has to start with 'use' keyword
- can have 0 or more arguments
- for function signature it is similar to normal functions, can have any no of arguments and any return type or no return at all.
