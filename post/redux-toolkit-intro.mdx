## Redux Toolkit
- mordern preferred way to use redux
- helps us to write less code 
- classic and toolkit way are compatable, can be written together

## Main 3 uses of Redux Toolkit
- we can write code that mutates the state inside reducer. it uses ```immer``` library to convert
mutable to immutable logic behind.
- automatically create actionCreator fn.
- automatically set up thunk middleware and devtools
